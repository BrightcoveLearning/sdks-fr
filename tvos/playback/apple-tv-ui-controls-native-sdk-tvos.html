    Contrôles de l'interface utilisateur Apple TV avec le SDK natif pour les TVOS

    Dans cette rubrique, vous découvrirez les contrôles de lecture fournis par le SDK Brightcove Player pour les tvOS.

    Présentation

    Lorsque vous créez des applications Apple TV à l'aide du SDK Native Brightcove pour TVOS, vous obtenez un ensemble de contrôles de lecture pour permettre aux utilisateurs de naviguer dans le contenu vidéo, de modifier le rapport d'aspect de la vidéo, d'obtenir des informations vidéo et de gérer les sous-titres et les paramètres audio.

    Les contrôles fournis par le SDK natif Brightcove pour les TVOS sont les suivants :

    • Barre de progression qui peut également afficher l'heure de l'horloge
    • Barre d'onglets supérieure pour les informations vidéo, les sous-titres et l'audio
    • Possibilité d'ajouter vos propres vues et contrôles personnalisés dans la barre d'onglets supérieure

    Les contrôles et gestes utilisés seront familiers aux utilisateurs, car ils sont similaires à ceux trouvés dans l'interface AVKit.

    Avec l'insertion d'annonces côté serveur Brightcove (SSAI), les utilisateurs verront un aspect personnalisé lors de la lecture des annonces.

    Pour plus d'informations sur l'utilisation du kit SDK natif, consultez la présentation du document Présentation du SDK Brightcove Player pour tvOS .

    Versions supportées

    Les commandes de l'interface utilisateur Apple TV sont disponibles avec les versions suivantes :

    Version du système d'exploitation

    TVOS 9+

    Version du SDK natif

    SDK natif Brightcove pour iOS version 6.3.0+

    Télécommande Siri

    La télécommande Siri vous permet de contrôler votre Apple TV. Voici quelques-unes des façons dont il peut être utilisé :

    • Demandez à Siri de trouver ce que vous voulez regarder.
    • Utilisez la surface tactile pour vous déplacer autour de l'écran, balayer, mettre en surbrillance et sélectionner des éléments.
    • Appuyez sur les boutons de commande pour lire et suspendre rapidement le contenu, régler le volume et accéder aux écrans d'accueil et de menu.

    Pour plus de détails, consultez le document Utiliser votre télécommande Siri Remote avec Apple TV d'Apple.

    Commandes de lecture

    La lecture est contrôlée à l'aide de gestes et d'une pression sur les boutons de l'Apple Siri Remote. Voici quelques-unes des actions que vous pouvez utiliser avec votre application. Pour plus de détails, reportez-vous à ce qui suit :

    • Commandes de lecteur TV intégrées pour TVOS
    • Utilisation de l'interface utilisateur du lecteur TV avec le SDK Brightcove Player pour TVOS

    Barre de progression

    Un simple clic sur le trackpad Apple Siri Remote afficher/masquera la barre de progression. Lorsque la lecture est suspendue, le panoramique vers la gauche et la droite navigue vers l'arrière et l'avant dans votre vidéo.

    Pour le nettoyage, l'utilisateur peut cliquer et maintenir enfoncé, puis déplacer vers la gauche ou la droite pour avancer ou revenir rapidement à travers une vidéo.

    Pendant la lecture de la vidéo, l'utilisateur peut double-cliquer sur le quart gauche du trackpad Siri Remote pour remonter 10 secondes ou double-cliquer sur le quart droit du trackpad pour avancer 10 secondes.

    Rapport d'aspect vidéo

    Un double clic sur le trackpad Apple Siri Remote permet de parcourir trois rapports d'aspect vidéo standard :

    • Letterboxing pour respecter le rapport d'aspect (vidéo complète affichée)
    • Plein écran avec rapport d'aspect respecté (vidéo peut être recadrée)
    • Vue étirée pour s'adapter au plein écran (rapport d'aspect ignoré)

    Barre d'onglets supérieure

    Le balayage vers le bas sur le trackpad Apple Siri Remote affichera le menu supérieur et masquera la vue de progression. Une fois visible, le balayage vers le haut masquera le menu supérieur.

    La barre d'onglets comporte les éléments par défaut suivants :

    • Informations
    • Sous-titres
    • Audio

    Vous pouvez modifier les étiquettes de texte associées à chaque élément de la barre d'onglets ou créer vos propres vues personnalisées dans la barre d'onglets.

    Informations

    La sélection de l'élément de menu supérieur Info affiche le titre, la description et la longueur associés à la vidéo actuellement chargée. Ces données sont extraites des métadonnées de la vidéo dans l'API de lecture, mais vous pouvez définir votre propre texte lorsque la vidéo se charge.

    Sous-titres

    Si vous sélectionnez l'élément de menu supérieur Sous-titres , vous pouvez afficher les légendes associées à la vidéo. Ici, vous pouvez gérer les sous-titres/légendes à afficher pendant la lecture.

    Audio

    La sélection de l'élément de menu supérieur Audio affiche les options audio disponibles pendant la lecture vidéo.

    Personnalisé

    Avec le SDK natif Brightcove pour TVOS, vous pouvez facilement ajouter vos propres vues dans la barre d'onglets. Vos vues peuvent contenir toutes les vues ou contrôles UIKit standard dont vous avez besoin.

    Publicité avec SSAI

    En plus de la lecture de contenu, nous avons ajouté un mode publicitaire lors de l'utilisation de Brightcove Server Side Ad Insertion (SSAI). Ce mode comprend les fonctionnalités suivantes :

    • Possibilité d'empêcher l'utilisateur de scrubrer les annonces passées
    • Afficher les commentaires sur le temps et les annonces restantes dans un break publicitaire

    DVR en direct

    Lorsque vous travaillez avec votre application Apple TV, le SDK Brightcove Native pour TVOS prend en charge les flux en direct et DVR en direct. DVR live permet aux utilisateurs d'effectuer les opérations suivantes :

    • Utilisez la barre de progression pour rechercher les moments passés
    • Retournez au moment actuel en direct pour regarder du contenu en temps réel

    Mise en œuvre de la lecture

    Pour lire un flux DVR en direct ou en direct à partir de votre application Apple TV, procédez comme suit :

    1. Commencez par l' exemple de code de base de l'Apple TV. Ouvrez le projet dans Xcode et ouvrez le fichier ViewController.swift .

    2. Dans la createTVPlayerView() fonction, définissez playerType la valeur sur live ou liveDVR.

      playerView = BCOVTVPlayerView(options: options)
if (playerView != nil) {
playerView!.frame = self.videoContainerView.bounds
playerView!.playerType = BCOVTVPlayerType.liveDVR;
self.videoContainerView.addSubview(playerView!)
}
    3. Localisez la viewDidLoad() fonction.
    4. L'appel à la createSampleTabBarItemView() fonction est facultatif et vous montre comment créer une vue personnalisée dans la barre d'onglets. Pour cet exemple, nous allons commenter l'appel à cette fonction.
    5. Commentez l'appel à la requestContentFromPlaybackService() fonction.

    6. Dans la viewDidLoad() fonction, avant le crochet de fermeture, ajoutez du code pour effectuer les opérations suivantes :

      override func viewDidLoad()
{
super.viewDidLoad()
// Do any additional setup after loading the view, typically from a nib.

createTVPlayerView()

//        createSampleTabBarItemView()

// Create and configure the playback controller
playbackController.delegate = self
playbackController.isAutoAdvance = true
playbackController.isAutoPlay = true

// Link the playback controller to the Player View
playerView?.playbackController = playbackController

//        requestContentFromPlaybackService()

// live stream
let url = URL(string: "your live stream url")

let video: BCOVVideo = BCOVVideo(hlsSourceURL:url )
self.playbackController.setVideos([video] as NSArray)
}

      • Créez un BCOVVideo objet à l'aide de votre URL de flux DVR en direct ou en direct de l'étape précédente.
      • Appelez la setVideos() fonction pour ajouter votre flux en direct à la file d'attente playbackControllerde lecture de la, qui est configurée pour démarrer automatiquement la lecture.
