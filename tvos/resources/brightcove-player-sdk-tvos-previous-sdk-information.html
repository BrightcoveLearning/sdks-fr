    SDK Brightcove Player pour TVOS : Informations sur le kit SDK précédent

    Cette rubrique présente les versions précédentes du Kit de développement logiciel (SDK) et des plug-ins pour Native Player SDK pour TVOS.
    Les liens suivants pour la version précédente du SDK Brightcove Native Player pour tvOS sont fournis à titre indicatif uniquement. Il est recommandé d'utiliser la dernière version du SDK avec les intégrations de plugin.

    Versions précédentes

    Le SDK Brightcove Player pour tvOS est contenu dans le kit SDK Player pour iOS. Les liens suivants concernent la version précédente du SDK Brightcove Native Player pour iOS.
