Aperçu

Les SDK natifs de Brightcove incluent Android, iOS et tvOS. Ils sont construits sur les frameworks suivants :

  • Android - Bibliothèques construites au-dessus du Projet Android ExoPlayer (ExoPlayer 2 v2.9.3) [1]
  • iOS/tvOS - Bibliothèques construites au-dessus du Cadre Apple AVFoundation

Sauf indication contraire, un « Oui » dans le tableau indique que la fonctionnalité est prise en charge pour toutes les versions de système d'exploitation actuellement prises en charge par le SDK natif. Pour obtenir une liste des versions de système d'exploitation prises en charge, consultez ce qui suit :

  • SDK natif pour les versions prises en charge par Android
  • SDK natif pour les versions prises en charge par iOS
  • SDK natif pour les versions prises en charge par tvOS
