    Exécution d'exemples d'applications pour le SDK natif pour Android

    Dans cette rubrique, vous apprendrez comment exécuter les exemples d'applications associées au SDK Native Brightcove pour Android. Ces exemples fournissent un bon point de départ pour créer vos propres applications.

    Présentation

    Le SDK natif Brightcove pour Android fournit une variété d'exemples d'applications pour vous aider à démarrer vos propres implémentations. Pour obtenir la liste complète des exemples, reportez-vous à la section suivante :

    • Exemples de joueur pour Android
    En plus des exemples d'applications, il existe également une collection d'extraits de code que vous pourriez trouver utiles. Ils peuvent être trouvés dans les extraits de code à l'aide du document Native SDK pour Android .

    Si vous êtes nouveau dans les kits SDK natifs Brightcove, consultez les vidéos Getting Started with the Native SDK .

    Cadre

    Les exemples d'applications Android prennent en charge la dernière version du SDK Native Brightcove et des plugins.

    Il est recommandé d'utiliser le framework ExoPlayer 2 pour vos applications afin d'implémenter les dernières fonctionnalités disponibles avec le SDK natif. Si vous n'utilisez pas actuellement ce framework, consultez le document Migration vers ExoPlayer 2 Framework .

    Installation des exemples d'applications

    Les exemples d'applications Android prennent en charge la dernière version du SDK Native Brightcove et des plugins.

    Actuellement, toutes les applications d'exemples Android doivent être installées ensemble en tant que bundle. La façon la plus simple de commencer est d'importer les exemples d'applications dans Android Studio. C'est ce que nous ferons dans les étapes suivantes.

    Pour plus d'informations sur les exemples et pour en savoir plus sur l'option Contrôle de version pour l'installation, consultez le fichier readme des exemples de lecteur Android .

    Pour importer les exemples d'applications, procédez comme suit :

    1. Si vous ne l'avez pas déjà, téléchargez et installez la dernière version d' Android Studio.
    2. Clonez les échantillons Player pour Android sur votre ordinateur.
    3. Dans Android Studio, sélectionnez Importer le projet.

    4. Dans la boîte de dialogue de sélection de fichier, sélectionnez le fichier build.gradle de niveau supérieur dans votre référentiel cloné local.

      Les exemples d'applications SDK natif pour Android ont été testés à l'aide du processus de génération Gradle jusqu'à la version 3.1.4 incluse.
    5. Dans Android Studio, vous devriez voir tous les exemples de projets d'application.

    Exécution des exemples d'applications

    Les exemples d'applications peuvent être exécutés à l'aide des éléments suivants :

    • Émulateur Android Studio
    • Android Studio à l'aide d'un périphérique physique connecté à votre ordinateur
      • L'appareil doit avoir Android OS 4.4.4 ou version ultérieure
      • Sur votre appareil, assurez-vous que le débogage USB est activé

    Pour exécuter un exemple d'application, procédez comme suit :

    1. Dans Android Studio, dans la liste de projets de gauche, développez le projet BasicSampleApp .
    2. Développez le dossier java et ouvrez le fichier MainActivity.java .
    3. Dans la liste déroulante Sélection de projet, sélectionnez BasicSampleApp.
    4. À droite de la liste déroulante de sélection de projet, cliquez sur le bouton Exécuter .
    5. Si un périphérique est connecté à votre ordinateur, vous devriez le voir dans la boîte de dialogue Sélectionner une cible de déploiement . Sinon, après avoir créé un périphérique virtuel, vous le verrez également répertorié ici. Sélectionnez votre appareil et sélectionnez OK.
    6. Vous devriez voir l'application s'exécutant sur votre appareil ou dans l'émulateur.

    Débogage

    Android Studio dispose d'un débogueur qui vous permet de définir des points d'arrêt, d'examiner les variables et d'évaluer les expressions lors de l'exécution. Pour déboguer votre application dans l'émulateur, procédez comme suit :

    1. Dans Android Studio, localisez la ligne de code où vous souhaitez interrompre l'exécution.
    2. Dans la gouttière gauche pour la ligne de code, cliquez pour ajouter un point d'arrêt.
    3. Sélectionnez le bouton de débogage pour démarrer l'exécution.

    4. Lorsque l'exécution s'interrompt à votre point d'arrêt, vous disposerez de plusieurs options pour inspecter les variables et les expressions, ainsi que d'aller parcour/sur/dans des lignes de code.

      Pour plus d'informations sur le débogage de vos applications, reportez-vous au Guide de l'utilisateur pour le débogage d'Android Studio.
