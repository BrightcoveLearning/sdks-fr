Des produits

Conçus en pensant aux développeurs de plates-formes natives, les kits de développement logiciel natifs Brightcove visent à minimiser les délais de mise sur le marché en suivant les dernières normes en matière de développement d'applications dans chaque environnement.

Les kits de développement logiciel Brightcove Native sont construits sur des frameworks de lecteur natifs. Cela vous permet de profiter de la vitesse, des performances et de la flexibilité du système d'exploitation natif et de l'étendre en une expérience vidéo complète et facile à intégrer. De plus, nos SDK natifs suivent les conventions établies pour la mémoire, le nommage et les modèles de conception afin de rendre le démarrage aussi simple que possible.

Les SDK natifs sont conçus pour s'intégrer aux types d'appareils suivants :

  • Appareils mobiles
  • TV connectée / Décodeurs

Appareils mobiles

Nous fournissons les SDK suivants pour les appareils mobiles, chacun spécialisé dans son propre objectif :

  • Kit de développement logiciel (SDK) natif Brightcove pour iOS
  • Kit de développement logiciel natif Brightcove pour Android

Kit de développement logiciel (SDK) natif Brightcove pour iOS

Le Brightcove Native SDK pour iOS suit les conventions iOS modernes. Ainsi, que vous l'intégriez à une application Objective-C établie ou à une toute nouvelle application Swift, vous devriez vous sentir comme chez vous.

Versions iOS prises en charge
Afin de profiter des dernières améliorations technologiques d'Apple, nous proposons actuellement une assistance pour :

  • iOS 14
  • iOS 13.7
  • iOS 12.5.3

Avec la sortie d'iOS 9 et Xcode 7, apprenez à utiliser une nouvelle fonctionnalité appelée Sécurité du transport des applications (ATS).

En utilisant une combinaison de mesures collectées en interne et de rapports de données d'utilisation des appareils iOS publics d'Apple, le Brightcove SDK pour iOS vise à atteindre la grande majorité des utilisateurs.

Si vous êtes un développeur à la recherche de plus de détails, lisez le Kit de développement logiciel (SDK) natif Brightcove pour iOS documentation technique.

Kit de développement logiciel natif Brightcove pour Android

Le Brightcove Native SDK pour Android vise à atteindre la grande majorité des utilisateurs tout en profitant des améliorations publiées dans les dernières versions d'Android.

Versions Android prises en charge
En équilibrant cette portée avec la possibilité de tirer parti des améliorations publiées par Google, nous proposons actuellement une assistance pour les versions suivantes d'Android :

  • 10,0
  • 9.0
  • 8.0, 8.1
  • 7.0, 7.1
  • 6.0

À l'aide d'une combinaison de métriques collectées en interne et du public de Google Données d'utilisation de l'appareil Android rapport, le Brightcove SDK pour Android vise à atteindre la grande majorité des utilisateurs.

Si vous êtes un développeur à la recherche de plus de détails, lisez le Kit de développement logiciel natif Brightcove pour Android documentation technique.
