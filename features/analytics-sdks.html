introduction

Grâce aux données analytiques, vous pouvez suivre le succès de vos vidéos et mieux comprendre les habitudes de vos spectateurs. Les SDK Brightcove Native Player pour iOS et Android facilitent l'intégration de la collection d'analyses dans votre solution vidéo.

Solutions analytiques :

  • Analyse du nuage vidéo
  • Adobe Analytics (Omniture)

Analyse du nuage vidéo

Les clients Video Cloud qui utilisent le service de catalogue auront des données d'analyse automatiquement collectées par notre propre système Brightcove Analytics. Ces données sont ensuite agrégées et affichées dans le module Video Cloud Analytics. Le tableau suivant met en évidence les points de données actuellement collectés dans les SDK :

Événement suivi SDK iOS SDK Android
Impression vidéo
Vue vidéo
Engagement vidéo
Vidéo terminée

En plus de ces événements, vous pouvez spécifier vos propres données personnalisées, configurables dans l'application, qui peuvent être envoyées et agrégées pour être visualisées par Brightcove Analytics.

Brightcove Analytics est uniquement disponible pour les clients Brightcove Video Cloud.

Adobe Analytics (Omniture)

Offrant un examen approfondi des données derrière votre contenu, Adobe Analytics est un nom puissant dans le secteur de l'analyse. Profitant de cette expertise, les équipes du Brightcove Native Player SDK sont ravies de proposer des intégrations avec Adobe Analytics sur les plateformes iOS et Android.

Le plug-in Omniture est disponible pour les clients Brightcove Video Cloud et Brightcove Player. De plus, le plug-in Omniture peut être utilisé en combinaison avec l'analyse intégrée de Brightcove pour les clients Brightcove Video Cloud.

iOS

Pour plus de détails sur l'ajout du plug-in Omniture à votre application, consultez ce qui suit :

  • Lecteur Omniture de base pour iOS

Android

Pour plus d'informations sur l'ajout du plug-in Omniture à votre application, procédez comme suit :

  1. Commencez par le Exemple d'application Omniture de base.

  2. Dans le app/res/layout/omniture_activity_main.xml fichier, remplacez ce qui suit

    <com.brightcove.player.view.BrightcoveVideoView

    avec ça

    <com.brightcove.player.view.BrightcoveExoPlayerVideoView

  3. Dans le build.gradle , localisez le bloc de code Omniture et remplacez-le par celui-ci :

    if (project.name.contains('Omniture')) {
  dependencies {
    implementation "com.brightcove.player:android-omniture-plugin:${anpVersion}"
    implementation files(libraryDirectory + "adobeMobileLibrary.jar")
    implementation "com.brightcove.player:exoplayer2:${anpVersion}"
  }
}

  4. Dans le MainActivity.java fichier, recherchez le onCreate() fonction. Où le BrightcoveVideoView est créé, remplacez-le par ceci :

    brightcoveVideoView = (BrightcoveExoPlayerVideoView) findViewById(R.id.brightcove_video_view);
  5. Vous êtes prêt à exécuter l'exemple d'application avec le framework ExoPlayer.
