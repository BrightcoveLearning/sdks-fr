    Applications TV connectés/décodeur

    Dans cette rubrique, vous découvrirez la télévision connectée avec les SDK natifs Brightcove.

    Introduction

    La télévision connectée continue de croître dans l'utilisation et la popularité. Avec Brightcove, vous pouvez atteindre le public dans leur salon sur des téléviseurs grand écran avec du contenu haute définition.

    Les kits SDK Native Player offrent des intégrations transparentes pour les contrôles personnalisés de l'interface utilisateur, la publicité, la protection du contenu (DRM), la lecture en direct, les analyses Brightcove et les mesures QoE. Pour plus d'informations sur ces fonctionnalités, consultez le document Brightcove Native SDK Features .

    Brightcove vous permet de créer des applications TV attrayantes avec les plates-formes suivantes :

    • Kit de développement logiciel (SDK) pour lecteur natif
    • Lecteur SDK natif pour Android
    • Votre propre plateforme

    Kit de développement logiciel (SDK) pour lecteur natif

    Avec le SDK Brightcove Native Player pour TVOS, vous pouvez créer des expériences vidéo attrayantes pour la dernière génération d'Apple TV. Le SDK natif vous permet de créer des applications vidéo centrées sur l'Apple TV à l'aide du contenu de votre bibliothèque Video Cloud et prend en charge FairPlay Streaming (FPS) pour la livraison sécurisée de votre contenu premium.

    L' exemple d'application SSAI de base avec TVOS vous aidera à démarrer en fournissant une base pour votre application TV.

     

     

    Cet exemple d'application vous permet d'inclure des publicités côté serveur pour incorporer des publicités dans votre contenu vidéo, fournissant un flux continu pour la lecture.

       

    Si vous êtes un développeur recherchant plus de détails, lisez la documentation technique du kit de développement logiciel (SDK) Brightcove Native Player pour TVOS .

    SDK du lecteur natif pour Android

    Développez votre audience en tirant parti des workflows basés sur BrightCove. Avec le SDK Brightcove Native Player pour Android, vous pouvez créer des applications centrées sur la vidéo pour les TV Android et Fire TV d'Amazon.

    Le SDK natif pour Android vous permet d'exploiter votre contenu haute définition à partir de votre bibliothèque Video Cloud, et il prend en charge Widevine Modular pour la livraison sécurisée de votre contenu premium.

    L' application de base Android Sample vous permettra de commencer en fournissant une base pour votre application TV.

     

     

    Lorsque vous êtes prêt à introduire des contrôles personnalisés dans votre application, consultez l' application Styled Controlled Sample pour obtenir des conseils. Dans cet exemple, vous pouvez ajouter une icône de paramètres pour contrôler l'affichage des légendes.

     

     

    Une fois que vous activez les sous-titres, vous les verrez apparaître en bas de la vidéo. Cela suppose que vous avez associé un fichier de sous-titres à votre vidéo.

     

     

    Si vous êtes un développeur et voulez en savoir plus, lisez la documentation technique SDK de lecteur natifs Brightcove pour Android.

    Votre propre plateforme

    Si vous souhaitez développer des applications Smart TV sur votre plateforme gagnée, utilisez les API de Brightcove pour les intégrer à votre bibliothèque Video Cloud. Les API ci-dessous sont les mêmes que celles utilisées par le lecteur Brightcove et les SDK natifs Brightcove.

    API de lecture

    Utilisez l'API de lecture pour récupérer les métadonnées de la vidéo et de la liste de lecture de votre bibliothèque Video Cloud. Cette demande renvoie également l'URL de lecture actuelle. Pour plus de détails, reportez-vous à ce qui suit :

    • Présentation de l'API de lecture

    API de collecte de données

    Utilisez l'API de collecte de données pour envoyer des mesures de lecture à Brightcove Analytics. Notez que cela est fait automatiquement pour vous lorsque vous utilisez le lecteur Brightcove ou les kits SDK natifs. Pour plus de détails, reportez-vous à ce qui suit :

    • Présentation de l'API de collecte de données
