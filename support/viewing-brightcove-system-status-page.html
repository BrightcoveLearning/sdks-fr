Brightcove surveille en permanence l'état de tous les services et applications Brightcove. S'il y a des interruptions de service, une note sera affichée sur la page d'état du système. Tous les avis d'entretien programmé seront également affichés sur cette page.

Pour accéder à la page d'état du système, visitez https://status.brightcove.com ou à partir de l'en-tête de navigation, cliquez sur l'icône Support () puis  État du système.

Toutes les notifications publiées comprendront les informations suivantes :

  • Statut de l'incident
    • Opérationnel (Vert)
    • Notification (Bleu - les systèmes sont actuellement en fonctionnement mais affectés par des retards ou des pannes intermittentes)
    • Perturbation partielle du service (jaune)
    • Perturbation du service complet (rouge)
  • Composants - Service ou zone affectée
  • Emplacements
    • APAC
    • Amériques
    • EMEA
    • Japon
  • État actuel et statut. Les États comprennent :
    • Enquête - Les ingénieurs se penchent sur le problème
    • Identifié - Les ingénieurs ont identifié le problème et cherchent à le corriger
    • Surveillance - Problème résolu et surveillance technique pour s'assurer que tout est sain
    • Résolu - Problème résolu

Tous les messages publiés seront en anglais et l'heure utilisée est l'heure normale de l'Est (GMT-5).

Affichage de l'historique de l'état du système

Pour afficher l'historique de l'état du système, cliquez sur le bouton Histoire lien en bas de page.

S'abonner aux changements d'état du système

Vous pouvez vous inscrire pour être averti lorsque l'état du système change. Cette section est organisée comme suit:

  • S'abonner à la messagerie
  • Abonnement au webhook
  • Gérer votre abonnement

S'abonner à la messagerie

Pour recevoir des notifications par e-mail, cliquez sur le S'ABONNER , entrez votre adresse e-mail puis cliquez sur S'abonner.

Abonnement au webhook

Un webhook est un rappel défini effectué avec HTTP POST. Vous spécifiez une URL où le rappel publiera un objet JSON en cas de mise à jour de statut.

Pour vous inscrire et être averti lorsque l'état du système change, sélectionnez le S'ABONNER bouton. Saisissez ensuite une URL pour recevoir la charge utile JSON avec votre adresse e-mail.

Gérer votre abonnement

Après vous être abonné, vous recevrez un e-mail avec un lien vers une page Gérer l'abonnement qui vous permet de gérer votre abonnement. Vous pouvez choisir les composants/produits/régions pour lesquels vous souhaitez recevoir des notifications. Faites vos sélections puis cliquez sur Enregistrer l'abonnement.

Pour vous désinscrire, cliquez sur le Se désabonner au bas de la page Gérer l'abonnement.

Recevoir des notifications dans Slack

Slack possède son propre système de webhook qui peut être utilisé pour connecter les mises à jour de statut. Consultez le Documentation Slack pour plus de détails.
