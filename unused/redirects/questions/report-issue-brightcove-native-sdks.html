Aperçu

Quelque chose s'est mal passé ou s'est produit de manière inattendue lors du développement avec les SDK Brightcove Native Player ? Faites-nous savoir ce qui s'est passé pour que vous puissiez reprendre le dessus.

Afin d'accélérer le processus de rapport, procédez comme suit avant de soumettre un bogue :

  1. Vérifiez s'il s'agit d'un problème connu
  2. Isoler le problème aux SDK Brightcove

Une fois que vous avez déterminé que votre problème n'est pas un problème connu et qu'il est isolé des SDK Brightcove, vous êtes prêt à soumettre un bogue. Veuillez suivre ces étapes :

  1. Fournir des informations spécifiques au cas
  2. Inclure des informations utiles supplémentaires
  3. Soumettez votre bogue

Vérifier si problème connu

Tout d'abord, vérifiez si votre problème a été signalé dans le forum des développeurs :

  • Forum des développeurs de Brightcove Player SDK

Ensuite, vérifiez si votre erreur est un problème connu indiqué dans la documentation du développeur :

  • Problèmes connus du SDK natif

Isoler le problème

Essayez de reproduire le problème dans l'un des exemples d'application fournis. Ces applications fournissent une implémentation de base et peuvent vous aider à déterminer si le problème provient du SDK ou d'un facteur externe.

Testez avec l'un des exemples d'applications suivants :

  • Exemples d'applications SDK Brightcove pour iOS
  • Exemples d'applications SDK Brightcove pour Android

Fournir des informations

Pour fournir des informations spécifiques au cas, rassemblez les éléments suivants :

  • Comportement d'erreur que vous rencontrez / le comportement attendu
  • Détails spécifiques à l'appareil (marque, version du modèle)
  • Version SDK utilisée
  • Journaux de plantage (logcat pour Android, journaux de plantage de la console/de l'appareil)
  • Étapes à reproduire

Information additionnelle

Incluez les détails suivants si vous le pouvez :

  • Extrait de code
  • Exemple d'application affichant une erreur
  • Ce que vous avez essayé de résoudre le problème

Soumettez votre bogue

Utilisez ce qui suit pour saisir votre bogue à l'équipe d'assistance :

  • Créer un ticket de support général
