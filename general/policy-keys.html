introduction

La clé de stratégie est utilisée pour récupérer le contenu vidéo et de liste de lecture de votre bibliothèque Video Cloud à l'aide de l'API de lecture. Étant donné que la clé de stratégie est définie sur le lecteur, elle vous permet de restreindre l'accès à votre contenu au niveau du lecteur. Vous pouvez également contrôler la disponibilité du contenu au niveau de la vidéo en utilisant le Module multimédia de Video Cloud Studio.

Lorsque vous créez un Brightcove Player, une clé de stratégie est automatiquement générée pour chaque lecteur. Cette clé est associée à votre compte Brightcove et contrôle quelles vidéos peuvent être visionnées et quand elles peuvent être visionnées, à condition que les vidéos soient stockées dans Video Cloud. Par exemple, des clés de stratégie sont nécessaires pour limiter l'accès en fonction de :

  • Compte
  • Restrictions de domaine, le cas échéant
  • Géo-restrictions, le cas échéant

Pour plus d'informations sur les clés de stratégie, consultez le Aperçu: API de règles document.

Remarques:
  • La clé de politique est uniquement utilisée pour contrôler les vidéos stockées dans Video Cloud. Si vous utilisez le Brightcove Player autonome, il n'est pas nécessaire d'accéder ou de récupérer la clé de stratégie.
  • Les clés de stratégie générées pour les nouveaux lecteurs Brightcove ne sont pas activées pour la recherche par défaut. Pour rechercher des vidéos à partir de l'API de lecture, vous devez utiliser l'API de stratégie pour créer une nouvelle clé de stratégie activée pour la recherche. Pour plus de détails, consultez le Présentation de l'API des règles document.

Obtention de la clé de stratégie

La clé de stratégie est automatiquement générée et gérée pour vous. Ainsi, dans la plupart des cas, vous n'aurez pas besoin de le demander ou de le gérer. Le moyen le plus simple de récupérer votre clé de stratégie est de procéder comme suit :

  1. Ouvrez le JOUEURS module et soit créer un nouveau lecteur, soit localiser le lecteur auquel vous souhaitez ajouter le plugin.
  2. Sélectionnez le lien du lecteur pour ouvrir les propriétés du lecteur.

  3. Dans le menu de navigation de gauche, sélectionnez Éditeur JSON.

  4. Dans le code JSON affiché, vous verrez la clé de stratégie :

Dans de rares cas, vous n'avez peut-être pas créé de Brightcove Player, mais vous avez toujours besoin de votre clé de stratégie (par exemple, vous développez strictement avec les SDK Brightcove Native Player). Dans ce cas, vous devrez contacter service client pour obtenir votre clé de stratégie.

Utilisation des clés de stratégie

La clé de stratégie peut être utilisée pour authentifier les demandes adressées à l'API Playback. L'API de lecture est destinée à être utilisée au moment de l'exécution dans des pages Web ou des applications mobiles pour la gestion générale des médias et l'intégration avec d'autres systèmes. Pour plus de détails, consultez le API de lecture sommaire.

Clés de stratégie par rapport aux informations d'identification de l'API

La clé de stratégie est utilisée pour accéder à l'API de lecture, tandis que les informations d'identification de l'API sont utilisées pour accéder à votre contenu dans les autres API Video Cloud.

Clés de stratégie

La clé de stratégie est utilisée pour la récupération côté client des actifs de votre bibliothèque Video Cloud. Voici les détails:

  • Accédez à vos données vidéo/playlist à l'aide de l'API de lecture
  • Lecture seulement
  • Vos données vidéo/playlist sont mises en cache sur le serveur par l'API Playback
  • Utilisez le API de règles pour faire ce qui suit :
    • Créez des clés de stratégie. Vous devez contacter le support client pour révoquer une clé de stratégie.
    • Définissez des restrictions de domaine et des restrictions géographiques au niveau de la stratégie. Les restrictions sont appliquées par les joueurs qui utilisent la clé de stratégie.
Vous pouvez également utiliser le Méthodes du catalogue de Brightcove Player pour récupérer le contenu de votre bibliothèque Video Cloud.

Identifiants de l'API

Les informations d'authentification API (également appelées informations d'identification client) permettent aux applications côté serveur de gérer les ressources de votre bibliothèque Video Cloud. Voici les détails:

  • Accédez à vos données à l'aide des API Video Cloud. Vous aurez besoin des informations d'identification client pour le compte et les API que vous souhaitez utiliser.
  • Lire, insérer, mettre à jour, supprimer
  • Vos données vidéo/playlist ne sont pas mises en cache sur le serveur par l'API CMS
  • Utilisation Interface utilisateur d'authentification API de Video Cloud pour créer et gérer les identifiants API

Gestion des clés de stratégie

Vous pouvez gérer vos clés de stratégie comme suit :

Obtention

Il n'existe aucun moyen de mettre à jour une clé de stratégie. Tu peux soit obtenir la clé de la politique à partir d'un lecteur existant dans votre compte Video Cloud, ou vous pouvez créer une clé de politique en utilisant le API de règles.

Révocation

Pour révoquer une clé de stratégie, vous devez contacter service client.

Les clés de politique ne sont PAS révoquées lorsqu'un lecteur est supprimé. Cela signifie que si vous utilisez une clé de stratégie d'un lecteur, puis supprimez le lecteur, vous pouvez toujours continuer à utiliser la clé de stratégie.
