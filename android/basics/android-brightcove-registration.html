Aperçu

Pour les clients Video Cloud, dans la plupart des cas, lorsque vous utilisez le Brightcove Player SDK pour Android, votre application est enregistrée automatiquement. Pour les clients Brightcove Player, vous devez envoyer votre identifiant de compte depuis votre application. Voir les détails ci-dessous.

Clients Cloud vidéo

Si vous utilisez le lecteur Brightcove et le Catalogue classe incluse dans le SDK pour récupérer et lire des vidéos, votre application sera automatiquement enregistrée auprès de Brightcove. En outre, les données d'analyse seront collectées et apparaîtront dans votre Video Cloud Analytics.

Si vous remplacez le BrightcoveVidéoVoir classe ou n'utilisez pas le lecteur et le catalogue Brightcove, vous devez envoyer votre ID d'éditeur Video Cloud à Video Cloud Analytics. Vous pouvez le faire en utilisant l'analyse setAccount() méthode. Cela vous permet également d'afficher les données de cette application dans Video Cloud Analytics.

Analytics analytics = brightcoveVideoView.getAnalytics();
analytics.setAccount("123456789");
Pour plus de détails sur l'envoi de données à Video Cloud Analytics, consultez le Utilisation de Brightcove Analytics guider.

Clients Brightcove Player

En tant que client Brightcove Player, vous devez enregistrer vos applications auprès de Brightcove. Pour ce faire, vous enverrez votre identifiant de compte à Brightcove à l'aide de l'analyse setAccount() méthode.

Analytics analytics = brightcoveVideoView.getAnalytics();
analytics.setAccount("123456789");
 

Maintenant que vous savez comment enregistrer votre application, vous êtes prêt à apprendre à travailler avec votre contenu multimédia.
