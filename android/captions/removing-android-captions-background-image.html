Aperçu

Lorsque vous chargez une vidéo avec des sous-titres dans une application créée avec le Brightcove Native Player SDK pour Android, un bouton de sous-titres apparaît dans le contrôleur multimédia.

Sélectionnez le bouton des sous-titres dans le contrôleur multimédia pour afficher une boîte de dialogue avec un Paramètres bouton.

Sélectionnez le Paramètres bouton pour afficher une image d'arrière-plan dynamique pour afficher les légendes pendant que vous les configurez.

Le fichier d'image d'arrière-plan des sous-titres a une taille de 400 ko. Si vous choisissez de vous débarrasser de cette image, vous pouvez la remplacer par un fichier image vide du même nom.

Supprimer l'image d'arrière-plan

Suivez ces étapes pour supprimer l'image d'arrière-plan des sous-titres par défaut :

  1. Dans le Kit de développement logiciel natif Brightcove pour Android , accédez au res/drawable-nodpi/caption_background.jpg déposer.
  2. Créez un fichier vide du même nom, caption_background.jpg.
  3. Remplacer l'existant caption_background.jpg fichier avec votre fichier vide du même nom.
  4. Recompilez votre projet.
