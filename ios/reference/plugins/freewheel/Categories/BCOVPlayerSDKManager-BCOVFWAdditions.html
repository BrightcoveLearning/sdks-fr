Référence de la catégorie BCOVPlayerSDKManager(BCOVFWAdditions)

Déclaré en BCOVFWComponent.h

Aperçu

Méthodes de catégorie ajoutées à BCOVPlayerSDKManager pour prendre en charge FW.

– createFWPlaybackControllerWithAdContextPolicy:viewStrategy :

Crée un contrôleur de lecture avec adContextPolicy et viewStrategy. Cette méthode utilise un plugin FreeWheel avec ses paramètres par défaut. Pour personnaliser le plug-in FreeWheel, utilisez createFWSessionProviderWithAdContextPolicy:upstreamSessionProvider:options.

- (id<BCOVPlaybackController>)createFWPlaybackControllerWithAdContextPolicy :(BCOVFWSessionProviderAdContextPolicy)adContextPolicy viewStrategy :(BCOVPlaybackControllerViewStrategy)stratégie

Paramètres

adContextPolicy

Le bloc qui construit le FWAdContext à utiliser par vidéo.
stratégie

La stratégie d'affichage de la propriété playbackController.view.

Valeur de retour

Un contrôleur de lecture.

Discussion

Crée un contrôleur de lecture avec adContextPolicy et viewStrategy. Cette méthode utilise un plugin FreeWheel avec ses paramètres par défaut. Pour personnaliser le plug-in FreeWheel, utilisez createFWSessionProviderWithAdContextPolicy:upstreamSessionProvider:options.

– createFWSessionProviderWithAdContextPolicy:upstreamSessionProvider:options :

Crée un fournisseur de session qui peut être utilisé pour créer un contrôleur de lecture.

- (id<BCOVPlaybackSessionProvider>)createFWSessionProviderWithAdContextPolicy :(BCOVFWSessionProviderAdContextPolicy)adContextPolicy amontSessionProvider :(id<BCOVPlaybackSessionProvider>)fournisseur options:(BCOVFWSessionProviderOptions *)options

Paramètres

adContextPolicy

Le bloc qui construit le FWAdContext à utiliser par vidéo.
fournisseur

Un autre fournisseur à utiliser, si nul, la valeur par défaut sera utilisée.
options

Les options pour personnaliser le plugin FreeWheel.

Valeur de retour

Un fournisseur de session.

Discussion

Crée un fournisseur de session qui peut être utilisé pour créer un contrôleur de lecture.
