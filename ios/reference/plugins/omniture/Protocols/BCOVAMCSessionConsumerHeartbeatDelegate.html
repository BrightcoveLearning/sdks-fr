Aperçu

Méthodes facultatives pouvant être implémentées par les délégués d'un utilisateur de session AMC pour l'analyse de pulsation vidéo Adobe.

– heartbeatVideoUnloadedOnSession :

Appelé quand ADB_VHB_PlayerDélégué a reçu onVideoUnloaded un message.

- (void)heartbeatVideoUnloadedOnSession :(id<BCOVPlaybackSession>)session

Paramètres

session

La session de lecture avec le client de la session AMC spécifiée pour l'analyse Adobe Video Heartbeat qui reçoit le onVideoUnloaded un message.
