– initWithPublisherId:applicationId:

Crée un BCOVFPSAthorizationProxy pour interagir avec fps.brightcove.com.

- (type d'instance nullable)initWithPublisherId:(NSString *)pubId applicationId :(chaîne NSString *)appId

Paramètres

pubId

L'identifiant de l'éditeur. Ne doit pas être nul.
appId

L'identifiant de l'application enregistré auprès de fps.brightcove.com. Peut être nul si vous n'utilisez pas fps.brightcove.com.

Valeur de retour

Un proxy initialisé.

Discussion

Crée un BCOVFPSAthorizationProxy pour interagir avec fps.brightcove.com.
