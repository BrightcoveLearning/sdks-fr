Aperçu

Objet de stratégie pour générer des IMAAdsRequests à utiliser par une session de lecture d'entrée donnée.

+ videoPropriétésVMAPAdTagUrlAdsRequestPolitique

Renvoie une règle de demande d'annonces qui vérifie l'objet BCOVVideo dans chaque session de lecture pour l'URL du tag d'emplacement publicitaire VMAP à demander. Si l'objet vidéo’s properties contient une entrée dont la clé est kBCOVIMAAdTag , la valeur de cette entrée est supposée être une chaîne NSString contenant l'URL du tag d'emplacement publicitaire VMAP. Cette URL sera utilisée pour créer une demande VMAP IMAAdsRequest pour cette session de lecture.

+ (type d'instance)videoPropertiesVMAPAdTagUrlAdsRequestPolicy

Valeur de retour

Une règle de demande d'annonces qui génère des demandes VMAP IMAAdsRequests à partir des informations de chaque session de lecture’s BCOVVideo.properties.

Renvoie une règle de demande d'annonces qui vérifie l'objet BCOVVideo dans chaque session de lecture pour l'URL du tag d'emplacement publicitaire VMAP à demander. Si l'objet vidéo’s properties contient une entrée dont la clé est kBCOVIMAAdTag , la valeur de cette entrée est supposée être une chaîne NSString contenant l'URL du tag d'emplacement publicitaire VMAP. La règle de progression des points de repère spécifiée pour les annonces VAST demande des points de repère. + adsRequestPolicyFromCuePointPropertiesWithAdTag:adsCuePointProgressPolicy :

Renvoie une politique de demande d'annonces qui vérifie chaque BCOVVideo pour les BCOVCuePoints de type ‘kBCOVIMACuePointTypeAd’. Les propriétés du point de repère seront ajoutées au tag d'emplacement publicitaire en tant que paramètres de requête. Cette URL sera utilisée pour créer une VAST IMAAdsRequest pour cette session de lecture.

+ (type d'instance)adsRequestPolicyFromCuePointPropertiesWithAdTag:(NSString *)étiquette publicitaire adsCuePointProgressPolicy:(BCOVCuePointProgressPolicy *)adsCuePointProgressPolicy

Paramètres

étiquette publicitaire

URL du tag de redirection à inclure dans la demande IMAAdsRequest générée par cette règle.
adsCuePointProgressPolicy

La règle de progression des points de repère spécifiée pour les annonces VAST demande des points de repère. Si ce paramètre est nul, +[BCOVCuePointProgressPolicy progressPolicyProcessingCuePoints:BCOVProgressPolicyProcessAllCuePoints resumingPlaybackFrom:BCOVProgressPolicyResumeFromContentPlayhead ignoringPreviouslyProcessedCuePoints:NO] sera utilisé par défaut.

Valeur de retour

Une règle de demande d'annonces qui génère des demandes VAST IMAAdsRequests à partir de étiquette publicitaire et des informations dans chaque vidéo de lecture’s BCOVCuePoint.properties.

Discussion

Renvoie une politique de demande d'annonces qui vérifie chaque BCOVVideo pour les BCOVCuePoints de type ‘kBCOVIMACuePointTypeAd’. Les propriétés du point de repère seront ajoutées au tag d'emplacement publicitaire en tant que paramètres de requête. Cette URL sera utilisée pour créer une VAST IMAAdsRequest pour cette session de lecture.

Utilisez cette règle lorsque vous souhaitez utiliser la même URL de tag d'emplacement publicitaire VAST pour tous les points de repère.
