– appropriéSourceNotFound

Cette méthode est appelée lorsqu'une source appropriée n'est pas trouvée.

- (void)appropriéSourceNotFound

Discussion

Les BCOVGoogleCastManager recherchera la version HTTPS de HLS v3, puis vérifiera DASH et enfin une source MP4. Si aucune version HTTPS n'est trouvée, il recherchera alors les versions HTTP des mêmes types de source. Si aucune version HTTP n'est trouvée, cette méthode sera appelée.

– devraitCastVideo :

Cette méthode est appelée après BCOVGoogleCastManager a reçu une nouvelle session de lecture.

- (BOOL)devraitCastVideo :(BCOVVideo *_Nonnull)vidéo

Discussion

Lorsque BCOVGoogleCastManager reçoit une nouvelle BCOVVideo d'une session de lecture, il tentera de la comparer à la BCOVVideo précédente, s'il y en a une, pour empêcher la session de diffusion de redémarrer avec la même vidéo. Vous pouvez utiliser cette méthode pour remplacer la vérification interne afin de contrôler si la vidéo actuelle doit être diffusée ou non.
