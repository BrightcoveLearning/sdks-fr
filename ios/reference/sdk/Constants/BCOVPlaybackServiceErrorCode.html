Référence des constantes BCOVPlaybackServiceErrorCode

Code d'erreur BCOVPlaybackService

Erreurs associées aux requêtes de l'API Playback.

Définition

typedef NS_ENUM(NSInteger, BCOVPlaybackServiceErrorCode ) {
   BCOVPlaybackServiceErrorNoError,
   BCOVPlaybackServiceErrorCodeConnectionError,
   BCOVPlaybackServiceErrorCodeJSONDeserializationError,
   BCOVPlaybackServiceErrorCodeAPIError,
} ;

Constantes

BCOVPlaybackServiceErrorNoError

Pas d'erreur.

Déclaré en BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeConnectionError

Code d'erreur indiquant qu'une erreur s'est produite lors de la connexion à l'API de lecture. Vérifiez la clé error.userInfo NSUnderlyingErrorKey pour l'erreur de connexion réelle.

Déclaré en BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeJSONDeserializationError

Code d'erreur indiquant qu'une erreur s'est produite lors de l'analyse de la réponse en tant que JSON. Si la demande a renvoyé des données, cette NSData brute sera incluse dans userInfo par la clé kBCOVPlaybackServiceErrorKeyRawResponseData. Vérifiez la clé error.userInfo NSUnderlyingErrorKey pour l'erreur de désérialisation JSON réelle.

Déclaré en BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeAPIError

Code d'erreur indiquant qu'une erreur a été renvoyée par l'API. Vérifiez la clé error.userInfo kBCOVPlaybackServiceErrorKeyAPIErrors pour un NSArray d'erreurs renvoyé par l'API et la clé kBCOVPlaybackServiceErrorKeyAPIHTTPStatusCode pour trouver le code d'état renvoyé.

Déclaré en BCOVPlaybackService.h.
