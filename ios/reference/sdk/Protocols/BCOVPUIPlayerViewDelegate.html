– playerViewShouldDisplayTumbnailAperçuAvecRect :

Appelé juste avant la configuration de l'aperçu miniature.

- (CGRect)playerViewShouldDisplayThumbnailPreviewWithRect:(BCOVPUIPlayerView *)joueurVoir

Paramètres

joueurVoir

La vue du lecteur qui affichera l'aperçu miniature.

Valeur de retour

Le CGRect personnalisé pour l'aperçu des vignettes

Discussion

Appelé juste avant la configuration de l'aperçu miniature.

Vous pouvez l'implémenter pour personnaliser la taille de l'aperçu des vignettes pour différents appareils, mises en page, modes d'écran, etc.

Les X la valeur du CGRect sera ignorée, vous pouvez donc conserver le X valeur fixée à 0.

Si cette méthode déléguée n'est pas implémentée, la taille de l'aperçu miniature sera de 100x56 avec un décalage vertical de -60.
