Aperçu

Classe de valeur représentant une liste de lecture dans l'écosystème Brightcove Player SDK. Une liste de lecture est une collection ordonnée de vidéos , ainsi que des métadonnées facultatives supplémentaires sur la collection sous la forme de Propriétés. Notez que parce que BCOVPlaylist implémente NSFastEnumeration, il peut être utilisé dans des méthodes qui prennent des paramètres NSFastEnumeration (tels que [BCOVPlaybackController setVidéos :]).

– mettre à jour: méthode requise

Renvoie une version modifiée de cette liste de lecture. Étant donné que les objets BCOVPlaylist sont immuables, une toute nouvelle BCOVPlaylist doit être créée même si une seule modification est nécessaire. Par conséquent, cette méthode fournit un moyen pratique d'obtenir une copie modifiable temporaire de cette instance et de la modifier (dans updateBlock).

- (instancetype)update:(void ( ^ ) ( id<BCOVMutablePlaylist> mutablePlaylist ))updateBlock

Paramètres

updateBlock

Un bloc auquel est transmise une copie modifiable de cette liste de lecture.

Valeur de retour

La copie de ce point de repère modifié par updateBlock.

Discussion

Renvoie une version modifiée de cette liste de lecture. Étant donné que les objets BCOVPlaylist sont immuables, une toute nouvelle BCOVPlaylist doit être créée même si une seule modification est nécessaire. Par conséquent, cette méthode fournit un moyen pratique d'obtenir une copie modifiable temporaire de cette instance et de la modifier (dans updateBlock).

N'essayez pas d'enregistrer une référence à la liste de lecture modifiable transmise à updateBlock , car le comportement de cette opération n'est pas défini. Au lieu de cela, apportez les modifications nécessaires dans le bloc, puis capturez l'objet de liste de lecture renvoyé par cette méthode.
