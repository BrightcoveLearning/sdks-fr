Référence du protocole BCOVSource

Conforme à NSObject
Déclaré en BCOVSource.h

Aperçu

Une spécificité “la source” ou la restitution dans un BCOVVidéo objet. Alors que Brightcove organise le contenu en entités appelées vidéos, les URL multimédias réelles auxquelles le contenu est accessible sont encapsulées par une source. En plus de l'URL, une source peut avoir un type de livraison facultatif qui peut être utilisé pour aider à la lever d'ambiguïté par rapport à d'autres sources, et un ensemble de Propriétés distincte de la Propriétés sur la vidéo qui en est propriétaire.

Notez qu'il est valable pour une vidéo de contenir plusieurs sources avec le même type de diffusion, auquel cas les sources doivent être désambiguïsées par d'autres moyens (comme avec des valeurs différentes dans leur Propriétés).

– mettre à jour: méthode requise

Renvoie une version modifiée de cette source. Étant donné que les objets BCOVSource sont immuables, un tout nouveau BCOVSource doit être créé même si une seule modification est nécessaire. Par conséquent, cette méthode fournit un moyen pratique d'obtenir une copie modifiable temporaire de cette instance et de la modifier (dans updateBlock).

- (instancetype)update:(void ( ^ ) ( id<BCOVMutableSource> mutableSource ))updateBlock

Paramètres

updateBlock

Un bloc auquel est passé une copie mutable de cette source.

Valeur de retour

La copie de cette source modifiée par updateBlock.

Discussion

Renvoie une version modifiée de cette source. Étant donné que les objets BCOVSource sont immuables, un tout nouveau BCOVSource doit être créé même si une seule modification est nécessaire. Par conséquent, cette méthode fournit un moyen pratique d'obtenir une copie modifiable temporaire de cette instance et de la modifier (dans updateBlock).

N'essayez pas d'enregistrer une référence à la source modifiable transmise à updateBlock , car le comportement de cette opération n'est pas défini. Au lieu de cela, apportez les modifications nécessaires dans le bloc, puis capturez l'objet source renvoyé par cette méthode.
