Déclaré en BCOVCuePoint.h

Aperçu

Classe de valeur représentant un point de repère dans l'écosystème Brightcove Player SDK. Un point de repère est utilisé pour spécifier un position sur la chronologie d'un élément de contenu qui peut être utilisé à n'importe quelle fin (souvent pour un contrôle programmatique sur le moment où les publicités doivent être affichées). Lorsque la lecture atteint un point de repère, la session de lecture envoie un événement sur son points de repère signal qui inclut la valeur BCOVCuePoint.

  position méthode requise

L'heure à laquelle le point de repère se produit sur la chronologie du BCOVVidéo qui le possède.

@property (non atomique, en lecture seule, assigner) position CMTime

Discussion

L'heure à laquelle le point de repère se produit sur la chronologie du BCOVVidéo qui le possède.

– mettre à jour: méthode requise

Renvoie une version modifiée de ce point de repère. Étant donné que les objets BCOVCuePoint sont immuables, un tout nouveau BCOVCuePoint doit être créé même si une seule modification est nécessaire. Par conséquent, cette méthode fournit un moyen pratique d'obtenir une copie modifiable temporaire de cette instance et de la modifier (dans updateBlock).

- (instancetype)update:(void ( ^ ) ( id<BCOVMutableCuePoint> mutableCuePoint ))updateBlock

Paramètres

updateBlock

Un bloc auquel est transmise une copie modifiable de ce point de repère.

Valeur de retour

La copie de ce point de repère modifié par updateBlock.

Discussion

Renvoie une version modifiée de ce point de repère. Étant donné que les objets BCOVCuePoint sont immuables, un tout nouveau BCOVCuePoint doit être créé même si une seule modification est nécessaire. Par conséquent, cette méthode fournit un moyen pratique d'obtenir une copie modifiable temporaire de cette instance et de la modifier (dans updateBlock).

N'essayez pas d'enregistrer une référence au point de repère modifiable transmis à updateBlock , car le comportement de cette opération n'est pas défini. Au lieu de cela, apportez les modifications nécessaires dans le bloc, puis capturez l'objet point de repère renvoyé par cette méthode.
