  Compte méthode requise

Cette propriété définira la valeur de l'ID de compte pour Brightcove Analytics. La définition de cette propriété remplacera également la valeur accountId sur toute vidéo récupérée via une requête API Brightcove Media.

  destination méthode requise

Cette propriété doit être un URI avec une structure valide et doit contenir une autorité. La valeur par défaut de cette propriété, si elle n'est pas remplacée, sera “bcsdk://” suivi de l'identifiant du bundle.

@property (non atomique, copie) NSString *destination

Discussion

Cette propriété doit être un URI avec une structure valide et doit contenir une autorité. La valeur par défaut de cette propriété, si elle n'est pas remplacée, sera “bcsdk://” suivi de l'identifiant du bundle.

Prière de se référer à http://en.wikipedia.org/wiki/URI_scheme#Generic_syntax pour plus d'informations et des exemples de syntaxe d'URI.

En particulier, une destination sans partie hiérarchique (par exemple juste un schéma) est considérée comme invalide, de même que toute valeur sans schéma.

  uniqueIdentifierActivé méthode requise

Cette propriété bascule la génération d'identifiant unique côté client. Si activé, le SDK identifiera les uniques utilisant l'appareil’s identifiant du fournisseur. S'il est désactivé, le SDK ne fournira aucune valeur unique et les analyses s'appuieront sur des heuristiques côté serveur pour déterminer les uniques.

@property (nonatomic, assign, getter=isUniqueIdentifierEnabled) BOOL uniqueIdentifierEnabled

Discussion

Cette propriété bascule la génération d'identifiant unique côté client. Si activé, le SDK identifiera les uniques utilisant l'appareil’s identifiant du fournisseur. S'il est désactivé, le SDK ne fournira aucune valeur unique et les analyses s'appuieront sur des heuristiques côté serveur pour déterminer les uniques.

La valeur par défaut est OUI.
