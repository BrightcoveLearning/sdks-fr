Aperçu

Une session de lecture représente la lecture d'un seul vidéo. La session fournit un point d'accès unique pour tout ce qui concerne l'expérience de lecture de cette vidéo : le vidéo et la source sélectionné pour la lecture, le joueur dans lequel la lecture se produit, et les événements qui se produisent pendant la session de lecture. Les sessions de lecture ne sont jamais réutilisées pour plusieurs vidéos (même des mises en file d'attente séparées du même vidéo).

  audibleMediaSelectionGroup méthode requise

L'AVMediaSelectionGroup des options de sélection de média audibles de l'AVAsset de l'AVPlayerItem de l'AVPlayer de la BCOVPlaybackSession en cours.

@property (non atomique, en lecture seule) AVMediaSelectionGroup *audibleMediaSelectionGroup

Discussion

L'AVMediaSelectionGroup des options de sélection de média audibles de l'AVAsset de l'AVPlayerItem de l'AVPlayer de la BCOVPlaybackSession en cours.

L'AVMediaSelectionGroup n'est pas trié et peut contenir des options non jouables. Utilisez la méthode de classe +playableMediaSelectionOptionsFromArray: de AVMediaSeledtionGroup pour renvoyer la liste des options de média lisibles. AVMediaSelectionGroup fournit des méthodes de filtrage pour, par exemple, ordonner la liste des options en fonction des préférences linguistiques de l'utilisateur.

– mettre fin méthode requise

Met fin à cette session de lecture, indiquant qu'une nouvelle session est prête à être retirée de la file d'attente. Une session de lecture terminée doit être supprimée immédiatement.

- (annuler) terminer

Discussion

Met fin à cette session de lecture, indiquant qu'une nouvelle session est prête à être retirée de la file d'attente. Une session de lecture terminée doit être supprimée immédiatement.

Dans une configuration typique utilisant un Contrôleur de lecture BCOV , il n'est pas nécessaire d'appeler cette méthode directement. Les méthodes sur le contrôleur sont le mécanisme préféré pour passer à la prochaine session de lecture. Cependant, les configurations de fournisseur de session de lecture qui n'utilisent pas de Contrôleur de lecture BCOV peut avoir besoin d'invoquer cette méthode pour faire avancer la file d'attente.
