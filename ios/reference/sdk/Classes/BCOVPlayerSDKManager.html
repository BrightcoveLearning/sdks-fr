Aperçu

Le Player SDK Manager est une instance singleton qui agit comme une usine, un registre central et un centre de diagnostic dans le Brightcove Player SDK pour iOS. Après avoir obtenu une instance du Manager, utilisez-la pour obtenir des instances d'autres objets de l'écosystème Player SDK.

Ajoute des méthodes à BCOVPlayerSDKManager pour créer des lecteurs FairPlay et des fournisseurs de session.

  ID de session

sessionID est un identifiant unique associé à cette instance de l'application. Cet identifiant est envoyé avec d'autres données d'analyse aux serveurs de mesures Brightcove.

@property (non atomique, en lecture seule, fort) NSString *sessionID

Discussion

sessionID est un identifiant unique associé à cette instance de l'application. Cet identifiant est envoyé avec d'autres données d'analyse aux serveurs de mesures Brightcove.

Le sessionID peut être utile lorsque vous essayez de localiser des problèmes avec des problèmes de lecture particuliers. Par exemple, si vous détectez des problèmes de lecture, vous pouvez envoyer l'ID de session et l'ID vidéo à votre propre serveur de suivi des erreurs, puis signaler ces deux valeurs aux ingénieurs de service Brightcove pour rechercher la source du problème dans les journaux du serveur.
