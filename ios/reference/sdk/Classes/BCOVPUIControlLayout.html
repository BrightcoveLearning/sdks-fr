Aperçu

BCOVPUIControlLayout L'objet BCOVPUIControlLayout est créé avec un ensemble de contrôles intégrés dans BCOVPUILayoutView objets avec quelques paramètres de personnalisation. L'objet BCOVPUIControlLayout est installé dans un BCOVPUILayoutView pour l'affichage dans une vue de joueur.

  éléments de mise en page standard

standardLayoutItems est le tableau contenant un ou plusieurs sous-tableaux de contrôles présentés lorsque la largeur de vue du lecteur est supérieure à compactLayoutLargeurMaximum. Chaque sous-tableau contient une liste de BCOVPUILayoutView objets qui seront présentés dans un BCOVPUILayoutView. Chaque sous-tableau représente une barre de contrôle dans la vue du lecteur. standardLayoutItems se compose généralement d'un seul sous-tableau représentant une seule ligne de contrôles.

  CompactLayoutItems

compactLayoutItems est le tableau contenant un ou plusieurs sous-tableaux de contrôles présentés lorsque la largeur de vue du lecteur est inférieure à compactLayoutLargeurMaximum. Chaque sous-tableau contient une liste de BCOVPUILayoutView objets qui seront présentés dans un BCOVPUILayoutView. Chaque sous-tableau représente une barre de contrôle dans la vue du lecteur. compactLayoutItems se compose généralement de deux sous-tableaux représentant deux rangées de contrôles, où chaque rangée a moins de contrôles pour mieux s'adapter à la largeur d'un écran iPhone.

Ensemble de tous les objets de mise en page. Les mêmes instances d'objet sont généralement utilisées dans les deux éléments de mise en page standard et CompactLayoutItems conteneurs.   compactLayoutLargeurMaximum

La largeur maximale des commandes compactes. Si la vue des contrôles est inférieure à cette valeur, le CompactLayoutItems sera présenté. S'il est plus petit, le éléments de mise en page standard sera utilisé.

– initWithStandardControls:compactControls :

Initialise une disposition BCOVPUIControlLayout avec les paramètres par défaut.

- (instancetype)initWithStandardControls:(NSArray *)StandardLayoutLines compactControls:(NSArray *)compactLayoutLines

Paramètres

StandardLayoutLines

Un tableau de tableaux, où chaque sous-tableau correspond à une ligne de commandes dans la vue des commandes du lecteur. Chacun de ces sous-tableaux est une liste d'objets BCOVPUILayoutView utilisés dans la disposition de largeur standard. Chaque vue de mise en page contient généralement un seul élément de contrôle et peut être créée avec [BCOVPUIBasicControlView layoutViewWithControlFromTag:width:elasticity:]
compactLayoutLines

Un tableau de tableaux, où chaque sous-tableau correspond à une ligne de commandes dans la vue des commandes du lecteur. Chacun de ces sous-tableaux est une liste de BCOVPUILayoutView objets utilisés dans la mise en page à largeur compacte (généralement, orientation portrait sur un iPhone).

Valeur de retour

Une instance de BCOVPUIControlLayout initialisée avec les contrôles spécifiés.

Discussion

Initialise une disposition BCOVPUIControlLayout avec les paramètres par défaut.
