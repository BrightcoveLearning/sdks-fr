  jouableHors ligne

Détermine si un rendu complet d'un AVAsset est disponible pour être lu sans connexion réseau.

@property (non atomique, en lecture seule) BOOL playableOffline

Valeur de retour

Renvoie YES si cette instance fait référence à un hors ligne vidéo lisible depuis l'appareil’s stockage local sans connexion réseau.

Renvoie NO si le téléchargement de la vidéo n'est pas terminé ou si la vidéo a été purgée et doit être retéléchargée. Cela se produit automatiquement si iOS a besoin de récupérer de l'espace de stockage pour le bon fonctionnement de l'appareil.

Sur certaines versions d'iOS jouableHors ligne peut retourner NON si le hors ligne la vidéo est déjà chargée dans une instance d'AVPlayer. Appel replaceCurrentItemWithPlayerItem sur l'instance actuelle de AVPplayer avec un néant valeur avant la vérification devrait entraîner le retour de la valeur attendue de TRUE.

Discussion

Interrogez cette propriété avant de présenter ou d'essayer de jouer un hors ligne actif pour déterminer s'il est disponible pour utilisation. Une valeur de retour de true indique qu'un rendu complet de cet actif est disponible pour hors ligne relecture.

La lecture de cette propriété est équivalente à la lecture de la propriété playableOffline de l'AVAssetCache d'un AVAsset.
