– resetHideControlsIntervalTimer

Réinitialise la minuterie utilisée pour savoir quand masquer les commandes. Si vous implémentez vos propres contrôles, vous pouvez l'appeler après que l'utilisateur les ait appuyés pour éviter que les contrôles ne se masquent trop tôt.

- (void) resetHideControlsIntervalTimer

Discussion

Réinitialise la minuterie utilisée pour savoir quand masquer les commandes. Si vous implémentez vos propres contrôles, vous pouvez l'appeler après que l'utilisateur les ait appuyés pour éviter que les contrôles ne se masquent trop tôt.
