Discussion

Cette méthode force toutes les tâches de téléchargement à se terminer avec une erreur en supprimant leur dossier de téléchargement sous-jacent. Les téléchargements déjà terminés ne sont pas affectés.

ATTENTION: Cette méthode ne doit être utilisée que dans des cas très spécifiques. iOS 11.0.x et iOS 11.1.x ont un problème où une tâche de téléchargement reprendra jusqu'à son terme si elle a été suspendue puis annulée. Cette méthode n'est pas garantie d'arrêter toutes les tâches de téléchargement.

paramètres

– addFairPlayApplicationCertificate:identifiant :

Cette méthode est utilisée pour ajouter un certificat d'application FairPlay à Offline Video Manager’s liste des certificats d'application FairPlay disponibles.

- (void) addFairPlayApplicationCertificate :(NSData *)applicationCertificatDonnées identifiant :(NSString *)identifiant

Paramètres

applicationCertificatDonnées

Le certificat d'application FairPlay dans un objet NSData. S'il est défini sur nil, tout certificat d'application existant pour l'identifiant donné sera supprimé.
identifiant

Une chaîne utilisée pour localiser le certificat d'application.

Cette chaîne ne peut pas être nulle.

Pour la livraison dynamique, l'identifiant doit être l'URL qui a été utilisée pour récupérer ce certificat.

Pour les comptes Video Cloud hérités, cela doit être défini sur kBCOVDefaultFairPlayApplicationCertificateIdentifier pour s'appliquer à tous les comptes.

Si vous utilisez plusieurs comptes Video Cloud hérités, définissez ce paramètre sur l'ID de compte.

Discussion

Les certificats seront conservés pour la durée de vie de la demande. Pour supprimer un certificat, ajoutez le même identifiant avec un nil que les données du certificat.

Si vous utilisez Dynamic Delivery, les certificats d'application sont récupérés automatiquement par le plug-in FairPlay, cette méthode n'est donc pas nécessaire. Vous pouvez toutefois utiliser cette méthode pour précharger votre certificat d'application afin d'accélérer la lecture de la première vidéo cryptée FairPlay.

Les certificats sont stockés et réutilisés pour les vidéos suivantes.
