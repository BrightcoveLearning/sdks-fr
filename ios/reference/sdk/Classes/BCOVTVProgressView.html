– addMarkerAt:duration:isAd:color :

Ajoutez un marqueur qui sera affiché sur le le progrès voir à un endroit précis.

- (void)addMarkerAt:(double)position durée :(double)durée isAd:(BOOL)estAnnonce couleur:(UIColor *)Couleur

Paramètres

position

L'emplacement (en secondes) dans le le progrès voir où le marqueur doit être placé. Voir le “durée” propriété pour les détails connexes.
durée

Les durée en secondes représentées par le marqueur ; peut être mis à zéro pour dessiner le plus petit marqueur de taille (un cercle).
estAnnonce

OUI indique que ce marqueur représente une annonce. NON indique qu'il s'agit d'un marqueur générique (dessiné en blanc par défaut). En règle générale, vous devez indiquer NON pour cette valeur, car les marqueurs publicitaires sont supprimés et rajoutés chaque fois que vous passez à une nouvelle session.
Couleur

Une UIColor qui sera utilisée pour dessiner le marqueur. Peut être mis à zéro.

Discussion

Ajoutez un marqueur qui sera affiché sur le le progrès voir à un endroit précis.
