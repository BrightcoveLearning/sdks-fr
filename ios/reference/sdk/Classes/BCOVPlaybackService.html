Hérite de NSObject
Déclaré en BCOVPlaybackService.h

Aperçu

La classe BCOVPlaybackService fournit des méthodes asynchrones pour récupérer des informations sur les vidéos à partir de l'API de lecture de Brightcove.

  SharedURLSession

NSURLSession partagée sur tous les appels réseau au BCOVPlaybackService. Cet objet est créé lors de l'initialisation du service de lecture. Si vous souhaitez utiliser votre propre NSURLSession, vous pouvez le définir ici. S'il est défini sur nil, la NSURLSession par défaut sera recréée.

@property (non atomique, fort) NSURLSession *sharedURLSession
