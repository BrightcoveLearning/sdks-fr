Aperçu

La politique de sélection de source de base décide quelle source sélectionner à partir d'un BCOVVidéo objet.

+ sourceSelectionHLSWithScheme :

Sélectionnez la première vidéo en HLS avec le schéma spécifié. Si aucune source de ce type n'est trouvée, la première source HLS avec n'importe quel schéma est renvoyée. Si aucune source HLS n'est présente, le premier MP4 est renvoyé. Si aucune des réponses ci-dessus n'est trouvée, la première source de quelque nature que ce soit est renvoyée.

+ (BCOVBasicSessionProviderSourceSelectionPolicy)sourceSelectionHLSWithScheme:(NSString *)schème

Paramètres

schème

Le schéma préféré (kBCOVSourceURLSchemeHTTP ou kBCOVSourceURLSchemeHTTPS) de l'URL source.

Valeur de retour

Une politique de sélection de source qui préfère HLS avec le schéma spécifié.

Discussion

Sélectionnez la première vidéo en HLS avec le schéma spécifié. Si aucune source de ce type n'est trouvée, la première source HLS avec n'importe quel schéma est renvoyée. Si aucune source HLS n'est présente, le premier MP4 est renvoyé. Si aucune des réponses ci-dessus n'est trouvée, la première source de quelque nature que ce soit est renvoyée.

Il s'agit de la politique de sélection de source par défaut (en utilisant kBCOVSourceURLSchemeHTTPS comme schéma).

+ sourceSélectionHLS

Sélectionnez la première vidéo en HLS, quel que soit le schéma. Si aucune source HLS n'est trouvée, sélectionnez la première source avec un deliveryType de “MP4”. Si ni l'un ni l'autre n'est trouvé, la première source de n'importe quel type est renvoyée.

+ (BCOVBasicSessionProviderSourceSelectionPolicy)sourceSelectionHLS

Valeur de retour

Une politique de sélection de source qui préfère HLS au MP4.

Discussion

Sélectionnez la première vidéo en HLS, quel que soit le schéma. Si aucune source HLS n'est trouvée, sélectionnez la première source avec un deliveryType de “MP4”. Si ni l'un ni l'autre n'est trouvé, la première source de n'importe quel type est renvoyée.
