  activée

Lorsque la propriété enabled est NO, la vue des éléments de la barre d'onglets n'est pas installée dans la barre d'onglets supérieure. Utilisez le BCOVTVParamètresAfficher méthode - (void)enable:(BOOL)enable tabBarItemView:(BCOVTVTabBarItemView *)tabBarItemView pour basculer simultanément le activée propriété et mettre à jour la barre d'onglets supérieure’s liste des UITabBarItems.

Cette propriété est utilisée pour définir le nom de l'UITabBarItem correspondant dans la barre d'onglets supérieure. Le titre doit être défini avant d'installer les vues des éléments de la barre d'onglets dans la vue Paramètres’s topTabBarItemViews biens.
